## Prerequisites
- Bash
- Docker
- Internet Access (only for initialization)

## How to run

To run in dev mode

```bash
./town-of-saviom-initializer.sh
```

To run with the frontend dockerized

```bash
./town-of-saviom-deploy.sh
```

With the frontend accessible at the URL <http://localhost>
